C

Claude Opus 4.5

기술 전문가 | Technical Expert

주요 강점

  • 정밀한 코드 분석 및 리뷰
  • 복잡한 버그 디버깅
  • 시스템 아키텍처 설계
  • 보안 취약점 탐지
  • 성능 최적화 제안
  • 200K 토큰 컨텍스트

최적 사용 사례

  • 💻 "이 코드의 버그를 찾아줘"
  • 🏗️ "마이크로서비스 아키텍처 설계"
  • 🔒 "보안 취약점 분석"
  • 📝 "코드 리뷰 및 리팩토링"

API 예시

from trinity import Trinity tri = Trinity() response = tri.ask( "이 Python 코드의 보안 취약점을 분석해줘", agents=["claude"] # Claude만 사용 )
G

Gemini 3 Pro

데이터 분석가 | Data Analyst

주요 강점

  • 종합적인 리서치 능력
  • 팩트체킹 및 검증
  • 이미지/비디오 멀티모달 분석
  • Deep Think 추론 모드
  • 시장 트렌드 분석
  • 데이터 시각화 제안

최적 사용 사례

  • 📊 "시장 데이터 분석해줘"
  • 🔍 "이 정보가 맞는지 검증해줘"
  • 🖼️ "이 이미지를 분석해줘"
  • 📈 "트렌드 리서치"

API 예시

from trinity import Trinity tri = Trinity() response = tri.ask( "2024년 AI 시장 트렌드를 분석해줘", agents=["gemini"] # Gemini만 사용 )

GPT 5.1

창의적 문제해결사 | Creative Solver

주요 강점

  • 브레인스토밍 및 아이디어 생성
  • 전략 수립
  • 통합 추론 능력
  • 내러티브 생성
  • 혁신적 사고

최적 사용 사례

  • 💡 "새로운 기능 아이디어"
  • 📋 "마케팅 전략 수립"
  • ✍️ "스토리 및 컨텐츠 작성"
  • 🎯 "비즈니스 전략 제안"

API 예시

from trinity import Trinity tri = Trinity() response = tri.ask( "새로운 SaaS 제품 아이디어를 브레인스토밍해줘", agents=["gpt"] # GPT만 사용 )
