Installation

# Clone repository git clone https://github.com/hwkim3330/agi.git cd agi # Install dependencies pip install -r requirements.txt # (Optional) Build WASM module cd wasm-consensus && wasm-pack build --target web && cd ..

Requirements

  • Python 3.10+
  • Rust 1.70+ (for WASM build)
  • API keys for Claude, Gemini, GPT

Quick Start

1. Set API Keys

export ANTHROPIC_API_KEY="your-anthropic-key" export GOOGLE_API_KEY="your-google-key" export OPENAI_API_KEY="your-openai-key"

2. Use CLI

# Basic question python trinity.py ask "마이크로서비스 아키텍처의 장단점" # With strategy python trinity.py ask "보안 취약점 분석" --strategy debate # Specific agents python trinity.py ask "코드 리뷰" --agents claude,gpt

3. Use Python API

from trinity import Trinity # Initialize tri = Trinity() # Basic usage response = tri.ask("REST vs GraphQL 비교") print(response.content) print(f"신뢰도: {response.confidence:.1%}") # With strategy response = tri.ask("신규 기능 아이디어", strategy="synthesis")

Configuration

config/agents.yaml 파일에서 에이전트 설정을 커스터마이즈할 수 있습니다.

# config/agents.yaml agents: claude: model: "claude-opus-4-5-20251101" max_tokens: 4096 temperature: 0.7 gemini: model: "gemini-3-pro" max_tokens: 4096 temperature: 0.7 gpt: model: "gpt-5.1" max_tokens: 4096 temperature: 0.7 consensus: default_strategy: "vote" timeout: 30

Trinity Class

Parameters

Parameter Type Default Description
config_path str None Path to custom config file

ask() Method

def ask( self, question: str, strategy: str = "vote", agents: List[str] = None, timeout: int = 30 ) -> TrinityResponse: """ Send question to AI agents and get consensus response. Args: question: The question to ask strategy: Consensus strategy (vote, synthesis, debate, specialist) agents: List of agents to use (default: all) timeout: Request timeout in seconds Returns: TrinityResponse object with consensus result """

Parameters

Parameter Type Default Description
question str required The question to ask
strategy str "vote" vote, synthesis, debate, specialist
agents List[str] None (all) ["claude", "gemini", "gpt"]
timeout int 30 Timeout in seconds

Response Object

@dataclass class TrinityResponse: content: str # Final consensus content confidence: float # Confidence score (0.0 - 1.0) strategy: str # Strategy used agents_used: List[str] # Agents that responded latency: float # Total latency in seconds metadata: Dict # Additional metadata

Example

response = tri.ask("Python 리스트 정렬 방법") print(response.content) # "리스트를 정렬하려면..." print(response.confidence) # 0.95 print(response.strategy) # "vote" print(response.agents_used) # ["claude", "gemini", "gpt"] print(response.latency) # 2.34

CLI Usage

# Basic python trinity.py ask "질문" # With options python trinity.py ask "질문" --strategy synthesis --agents claude,gpt # Help python trinity.py --help

CLI Options

Option Short Description
--strategy -s Consensus strategy
--agents -a Comma-separated agent list
--timeout -t Request timeout (seconds)
--verbose -v Verbose output
--json -j Output as JSON
