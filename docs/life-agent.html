Vision-Based Analysis

Supported Actions

  • CLICK - Click at X, Y coordinates
  • CLICK_CAPTCHA - Click CAPTCHA checkbox
  • TYPE - Click + Type text + Enter
  • SCROLL - Scroll up or down
  • WAIT - Wait for page load
  • DONE - Goal achieved

Vision Model Prompt

Analyze this browser screenshot. Current goal: {goal} Respond with: ACTION: CLICK / TYPE / SCROLL / DONE X: [x coordinate] Y: [y coordinate] TARGET: [what to click]
