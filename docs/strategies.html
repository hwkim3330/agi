Consensus Strategies

여러 AI의 응답을 어떻게 통합할지 결정하는
4가지 합의 전략을 제공합니다

전략 비교

전략 설명 사용 사례 신뢰도
Vote 최고 점수 응답 선택 정답이 명확한 질문 High
Synthesis 모든 응답 통합 종합적 분석 필요 Medium
Debate AI간 토론 후 결론 논쟁적 주제 High
Specialist 전문가 자동 선택 특화된 질문 Variable
🗳️

Vote Strategy

최고 점수 응답 선택

동작 방식

  1. 각 에이전트의 응답을 점수화
  2. 콘텐츠 길이, 응답 시간, 에이전트 가중치 고려
  3. 최고 점수 응답을 최종 결과로 선택

최적 사용

  • 정답이 하나인 질문
  • 기술적 문제 해결
  • 빠른 결과가 필요한 경우
# CLI python trinity.py ask "Python에서 리스트 정렬 방법" --strategy vote # Python API response = tri.ask("Python에서 리스트 정렬 방법", strategy="vote")
점수 계산 공식: score = base_score * content_factor * time_factor * agent_weight
🔄

Synthesis Strategy

모든 응답 통합

동작 방식

  1. 모든 성공한 응답 수집
  2. 각 에이전트의 역할 라벨 부여
  3. 구분선으로 분리된 통합 문서 생성

최적 사용

  • 다양한 관점이 필요한 분석
  • 복잡한 문제의 종합적 이해
  • 의사결정을 위한 정보 수집
# CLI python trinity.py ask "마이크로서비스 vs 모놀리식 아키텍처" --strategy synthesis # Python API response = tri.ask("마이크로서비스 vs 모놀리식", strategy="synthesis")
출력 형식: 
## 기술 전문가 (Claude)
[Claude의 분석]

---

## 데이터 분석가 (Gemini)
[Gemini의 분석]

---

## 창의적 문제해결사 (GPT)
[GPT의 분석]
💬

Debate Strategy

AI간 토론 후 결론

동작 방식

  1. 초기 응답 수집
  2. 각 AI가 다른 AI의 응답을 평가
  3. 강점/약점 분석 후 개선된 답변 생성
  4. 여러 라운드 반복 가능

최적 사용

  • 논쟁적인 주제
  • 찬반 의견이 필요한 경우
  • 심층적 분석이 필요한 문제
# CLI python trinity.py ask "AI 규제의 필요성" --strategy debate # Python API response = tri.ask("AI 규제의 필요성", strategy="debate")
토론 프롬프트 구조: 
원래 질문: {question}

다른 AI의 응답들:
[Claude]: {claude_response}
[Gemini]: {gemini_response}

위 응답들을 검토하고:
1. 각 응답의 강점과 약점을 분석하세요
2. 누락된 중요한 관점이 있다면 추가하세요
3. 가장 정확하고 완전한 최종 답변을 제시하세요
🎯

Specialist Strategy

전문가 자동 선택

동작 방식

  1. 질문에서 키워드 추출
  2. 각 에이전트의 전문 분야와 매칭
  3. 가장 적합한 전문가 선택
  4. 매칭 없으면 모든 응답 통합

최적 사용

  • 특화된 전문 영역 질문
  • 빠른 전문가 의견이 필요할 때
  • 자동 라우팅을 원할 때
# CLI python trinity.py ask "이 Python 함수 디버깅 해줘" --strategy specialist # Python API response = tri.ask("이 Python 함수 디버깅 해줘", strategy="specialist")

Gemini

연구, research, 논문, 데이터, 분석, 통계, 팩트, fact, 검증, 이미지, 비디오, 멀티모달, 시장, 트렌드

GPT

창의, creative, 아이디어, 브레인스토밍, 전략, strategy, 기획, 마케팅, 스토리, 글쓰기, writing, 컨텐츠, 혁신

전략 선택 가이드

1

정답이 명확한 질문

→ Vote 사용

예: "Python에서 리스트 정렬하는 방법"

2

다양한 관점이 필요한 분석

→ Synthesis 사용

예: "마이크로서비스 아키텍처의 장단점"

3

논쟁적이거나 복잡한 주제

→ Debate 사용

예: "AI 규제의 필요성"

4

특화된 전문 영역

→ Specialist 사용

예: "이 코드의 보안 취약점 분석"
